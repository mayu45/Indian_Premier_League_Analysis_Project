# Project Title
Indian_Premier_League_Analysis_Project


# Project Description
This project is an Excel-based Interactive Dashboard built to analyze Indian Premier League (IPL) performance across multiple seasons.It includes slicers, charts, and summary metrics to help users exploreteams, winners, matches, and trends from 2008 onwards.The dashboard is fully automated using formulas, pivot tables, and advanced Excel visualization techniques.


# Key Features (Points for README)
## 1. Dashboard Highlights
•	Interactive Season-wise performance overview <Br /> 
•	Team-wise comparison using bar charts <Br />
•	Toss decision analysis (Bat/Field trend) <Br />
•	Top 10 Match of the Match winners <Br />
•	Win/Loss charts for each season <Br />
•	Dynamic cards showing: <Br />
o	Season <Br />
o	Winner <Br />
o	Runner-up <Br />
o	Player of the Series <Br />

## 2. Technical Skills Used
•	Excel Pivot Tables <br />
•	Excel Pivot Charts <br />
•	Slicers & Filters <br />
•	Data Cleaning & Formatting <br />
•	Conditional Formatting <br />
•	Formulas (SUM, COUNT, IF, etc.) <br />
•	Dashboard Layout & Design Techniques <br />

## 3. Dataset Information
•	IPL match data from 2008 to 2019 <br />
•	Includes: Match results, teams, toss decision, winners, awards <br />

## 4. Objective
•	To create a modern and interactive Excel dashboard that provides quick insights into IPL match history and helps viewers understand season trends. <br />

## 5. Outcome / Achievement

•	Clear visualization of trends like: <br />
